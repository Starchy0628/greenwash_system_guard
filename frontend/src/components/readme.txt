============================================================
frontend/src/components - Vue组件目录
============================================================

【文件夹作用】
所有Vue单文件组件，按功能模块组织。

【核心组件说明】
- HeroBanner.vue       首屏搜索区域 + 结果展示（合并组件）
- PdfUpload.vue        PDF拖拽上传组件
- ResultCard.vue       企业分析结果卡片（GW指数环形图、风险等级）
- SentenceList.vue     语句级分类结果列表（三类Tab切换）
- ChinaMap.vue         中国省份风险地图（ECharts可视化）
- Top10Grid.vue        Top10高风险企业卡片网格
- Top10Card.vue        单个企业风险卡片
- TrendChart.vue       企业历史GW指数趋势图
- MarketTrendChart.vue 市场整体趋势图表
- MethodSteps.vue      方法学说明步骤展示
- GradingRule.vue      评分规则说明
- WatchList.vue        关注列表面板
- AppHeader.vue        顶部导航栏
- BrandLogo.vue        品牌Logo组件
- SealTag.vue          风险标签/印章组件
- SectionTitle.vue     章节标题组件
- StatCard.vue         统计数据卡片
- ToastNotification.vue 消息提示组件
- LiveSteps.vue        实时分析步骤进度展示
- index.js             组件统一导出
