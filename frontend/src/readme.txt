============================================================
frontend/src/ - 前端源代码
============================================================

【文件夹作用】
Vue 3前端源代码，包含组件、状态管理、API封装。

【子文件夹说明】
- api/           Axios HTTP请求封装
- components/    Vue组件（搜索、结果、地图、图表等）
- stores/        Pinia状态管理
- assets/        静态资源（图片等）
- composables/   Vue组合式函数

【关键文件】
- main.js        Vue应用入口（注册Pinia、Naive UI等）
- App.vue        根组件
