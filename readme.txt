============================================================
谛观 GreenwashGuard - 企业漂绿风险智能监测平台
============================================================

【文件夹作用】
项目根目录，包含整个参赛作品的所有文件。
基于三异构大语言模型多数投票机制的A股上市公司漂绿风险测算平台。

【文件/子文件夹说明】
- backend/              FastAPI后端服务代码（含API、算法、数据库）
- frontend/             Vue 3前端界面代码（含预构建产物dist/）
- data/                 可选数据目录（放置CMDA管理层讨论与分析文本）
- 启动系统.vbs           [推荐] Windows一键静默启动，无黑框，自动开浏览器
- 启动系统.bat           开发者调试启动（显示控制台日志）
- 停止系统.vbs           一键停止后台服务
- launcher.py           Python启动管理器（VBS调用）
- launcher.log          启动日志（自动生成，排查问题用）
- README.md             项目快速上手指南
- 技术文档.md            详细技术文档（含算法、架构、实验、部署）
- .gitignore            Git版本控制忽略规则
- .gitattributes        Git属性配置

【启动方式（推荐）】
Windows环境下双击"启动系统.vbs"即可：
1. 自动检查Python环境
2. 自动安装依赖（首次）
3. 自动初始化数据库（首次）
4. 后台静默启动服务
5. 自动打开浏览器访问 http://localhost:8000

停止服务：双击"停止系统.vbs"

【环境要求】
- Python 3.10+（必需）
- 浏览器 Chrome/Edge/Firefox
- Node.js 不需要（前端已预构建到frontend/dist/）
