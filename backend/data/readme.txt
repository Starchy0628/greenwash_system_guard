============================================================
backend/data - 静态数据文件
============================================================

【文件夹作用】
存放静态配置数据，包括股票列表、行业分类映射等基础数据。

【文件说明】
- em_all_stocks.json          东方财富A股股票列表（代码、名称、行业）
- industry_map.json           申万一级行业分类映射
- st_codes_em.json            ST/*ST/PT股票代码列表（分析时排除）
- sw_industry_map_akshare.json  AKShare获取的申万行业映射
